# Branch Playbook: feat/mission-control

## Mission
Build mission-control dashboard for orchestration, monitoring, and ops controls.

## Agent Roster
- Primary: Frontend Developer
- Secondary: AI Engineer
- QA: Reality Checker

## First Task Queue
1. Dashboard shell + role-based sections
2. Live status cards (agents/jobs/health)
3. Action panel for safe operations
4. Audit trail/event timeline

## Acceptance Baseline
- Dashboard renders core system status
- Critical actions require confirmations
- Event timeline is queryable/readable
- No destructive action without safeguards
