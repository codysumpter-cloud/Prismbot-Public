# Branch Playbook: feat/prompt-forge

## Mission
Build Prompt Forge MVP (prompt builder + template library + export/share basics).

## Agent Roster
- Primary: Frontend Developer
- Secondary: AI Engineer
- QA: Reality Checker
- Growth handoff: Social Media Strategist

## First Task Queue
1. Prompt worksheet form UI
2. Template category browser
3. Copy-to-clipboard + markdown export
4. Prompt quality checks (Promptfoo wiring)

## Acceptance Baseline
- User can build prompt from structured fields
- User can save/copy/export prompt output
- Core templates load and render correctly
- Basic eval test exists for at least 3 templates
