# Prism Arcade Hub

## Vision
Old-school arcade vibe with modern streamer-friendly competition.

## MVP features
- Retro game launcher page
- Unified leaderboard
- Player profiles
- Weekly challenge rotation

## Streamer features
- Stream-safe HUD overlay
- "Beat my score" challenge links
- On-screen alerts for new top scores

## Tech suggestion
- Next.js frontend
- Supabase/Postgres leaderboard backend
- WebSocket updates for live rankings
