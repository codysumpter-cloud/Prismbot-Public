# Pixel Pets Arena

## Vision
Raise generated creatures, evolve them, and battle others.

## Core loop
- Hatch creature
- Feed/train/play
- Gain stats + traits
- Battle in async arena
- Earn cosmetics/titles

## MVP features
- Creature profile + stats
- Daily care actions
- Turn-based battle prototype
- Local leaderboard

## Stretch goals
- PvP seasons
- Breeding/traits
- Streamer challenge mode
