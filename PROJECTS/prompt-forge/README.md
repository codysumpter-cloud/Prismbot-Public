# Prompt Forge

## Vision
Make prompting easy for everyone with guided templates and copy-ready outputs.

## MVP features
- Prompt worksheet builder (role, goal, constraints, output format)
- Category templates (ads, coding, art, strategy)
- Save/favorite prompts
- Export to Markdown/PDF

## Tech suggestion
- Next.js + TypeScript
- Tailwind + shadcn/ui
- SQLite (or Supabase later)

## Milestones
1. Basic form builder
2. Template library
3. Export + share links
4. User accounts (optional)
