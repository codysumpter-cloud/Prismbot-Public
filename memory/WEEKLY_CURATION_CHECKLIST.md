# Weekly Memory Curation Checklist

Run weekly to keep memory sharp and compact.

## 1) Review recent notes
- Read last 7 daily memory files
- Extract key decisions, preferences, and outcomes

## 2) Promote important items
- Add durable decisions to `memory/decisions/`
- Add stable preferences to `memory/preferences/`

## 3) Prune noise
- Remove stale or duplicate entries
- Keep long-term memory concise

## 4) Validate alignment
- Ensure current plans match latest decisions
- Ensure assistant behavior reflects preference locks

## 5) Record curation pass
Create/update `memory/curation-log.md` with:
- Date
- Files reviewed
- Items promoted
- Items removed
- Follow-ups
