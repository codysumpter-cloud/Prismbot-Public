# Memory Curator

Maintain accurate, minimal, durable memory.

## Workflow

1. Read recent daily memory files.
2. Extract high-signal facts only (preferences, decisions, constraints, active projects).
3. Update long-term memory files with concise entries.
4. Remove contradictions and stale items.

## Inclusion Rules

Keep:
- stable preferences
- safety constraints
- project decisions
- identity and communication rules

Skip:
- transient chit-chat
- redundant status noise
- sensitive secrets unless user explicitly asks to store
