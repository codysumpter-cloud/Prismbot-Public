# Prismtek Professor Workbook

A practical roadmap to become an independent builder.

## How to use this workbook

- Do 45-90 minutes per session.
- Build while you learn (no passive studying).
- Ship tiny projects weekly.
- Use GitHub for everything.

---

## Module 1 — Builder Mindset + Learning System

### Core idea
You don’t need to know everything. You need a repeatable loop:
1) understand problem
2) build smallest version
3) test
4) improve
5) ship

### Exercise
Write your current top 3 project ideas and define a 7-day mini version for each.

---

## Module 2 — Bash Essentials

### Learn
- file navigation: `pwd`, `ls`, `cd`
- file operations: `mkdir`, `touch`, `cp`, `mv`
- search and inspect: `cat`, `less`, `grep`, `find`
- process basics: `ps`, `kill`

### Drill
Create a folder, add files, search for a keyword with `grep`, then archive the folder.

---

## Module 3 — Git + GitHub

### Learn
- local workflow: `status`, `add`, `commit`, `log`
- branch workflow: `checkout -b`, `merge`, `rebase`
- remote workflow: `push`, `pull`, PRs, issue tracking

### Golden workflow
Issue -> branch -> commit small -> PR -> review -> merge.

### Drill
Create one issue, one branch, one PR for a small README improvement.

---

## Module 4 — HTML/CSS Fundamentals

### Learn
- semantic HTML (header/main/section/footer)
- layout basics (flexbox, grid)
- responsive design (mobile first)
- accessibility basics

### Drill
Build a single-page profile site with a project grid.

---

## Module 5 — JavaScript + Node.js Basics

### Learn
- variables, arrays, objects, functions
- async/await and fetch
- npm packages
- simple Node server

### Drill
Build a Node script that reads a JSON file and prints a formatted report.

---

## Module 6 — React Fundamentals

### Learn
- components, props, state
- events and forms
- conditional rendering
- effects and data fetching

### Drill
Build a prompt builder page with live preview.

---

## Module 7 — React Router + App Architecture

### Learn
- route structure
- nested layouts
- 404 and error pages
- UI state patterns

### Drill
Split your app into Home, Builder, Library, and Settings pages.

---

## Module 8 — Vibe Coding (Pro Mode)

### Learn
- prompt structure (role, goal, constraints, output format)
- ask for tiny tasks and diffs
- verify each result before trusting it

### Rules
- Never merge code you didn’t run.
- Always ask for verification commands.
- Keep commits scoped to one change.

### Drill
Take one feature and implement it in 3 AI prompts max.

---

## Module 9 — Debugging + grep Mastery

### Learn
- reproduce bug first
- isolate root cause
- minimal fix + regression test
- grep patterns across files/logs

### Drill
Find all TODOs and all API calls in your project with grep/find.

---

## Module 10 — Prompt Engineering + Evaluation

### Learn
- quality prompts with strict output schema
- eval loops with Promptfoo
- red-team checks for safety and reliability

### Drill
Create 5 prompt tests and fail one on purpose, then fix it.

---

## Module 11 — AI Generation (Images + Assets)

### Learn
- prompt specificity (subject, style, constraints)
- iterative generation workflow
- consistency through templates and seeds

### Drill
Generate a character pack + tileset pack with a shared style guide.

---

## Module 12 — RAG Basics (Retrieval-Augmented Generation)

### Learn
- store docs/chunks
- retrieve relevant context
- feed context into model responses

### Drill
Build a mini RAG over your own playbooks and ask project questions.

---

## Module 13 — Machine Learning Path (Practical)

### Learn
- don’t train from scratch early
- start with adapters/LoRAs
- keep eval benchmarks fixed

### 12-month arc
1. collect clean rated data
2. add eval pipeline
3. train small adapters
4. compare quality and cost
5. scale only when justified

---

## Module 14 — Open Source Project Building

### Learn
- clear README + contribution guide
- issue templates + milestones
- release notes + changelog discipline

### Drill
Open-source one utility from your stack and onboard one contributor.

---

## Module 15 — Arcade/Product Vision Execution

### What to build
- Prompt Forge
- Pixel Pets Arena
- Arcade Streamer Hub

### Strategy
- ship MVP fast
- collect usage feedback
- iterate weekly

- Week 1-2: Bash + GitHub + HTML/CSS
- Week 3-4: JS + Node
- Week 5-6: React + Router
- Week 7: Debugging + grep + tests
- Week 8: Prompt engineering + Promptfoo
- Week 9: AI image pipeline
- Week 10: RAG basics
- Week 11: ML adapter prep
- Week 12: ship and publish one complete project

1. 10 min review

## Professor Rules (how I’ll coach you)

- explain like beginner first
- then show real-world pattern
- one exact next step at a time when debugging
- no fluff, no shame, always progress

---

## Graduation Criteria

You are “independent builder ready” when you can:
- create and ship projects without hand-holding
- use branches/PRs correctly
- debug and fix issues systematically
- evaluate prompts/models with evidence
- publish and maintain at least one open-source project
