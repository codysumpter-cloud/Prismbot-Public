# Weekend Bootcamp (Noob-Friendly)

## Day 1 — Bash + GitHub + First Ship

### Block A (60-90 min)
- Install/verify: git, node, npm
- Create a repo folder
- Run:
  - `git init`
  - `git add .`
  - `git commit -m "init"`

### Block B (60-90 min)
- Create first app with Vite React
- `npm create vite@latest my-first-app -- --template react`
- `cd my-first-app && npm install && npm run dev`

### Block C (45 min)
- Push to GitHub
- Create README with:
  - what it is
  - how to run
  - next steps

## Day 2 — React + Next + Real Mini Project

### Block A
- React basics: components, props, state
- Build one interactive component

### Block B
- Next.js intro + routes + layout
- Scaffold portfolio shell

### Block C
- Ship one tiny feature publicly
- Open first issue and milestone for next steps

## Definition of done
- 1 public repo live
- 1 local app running
- 3 clear next tasks in GitHub Issues
