# PrismBot Web (No Wix)

Standalone static website (no Wix dependency).

## Run locally

```bash
cd apps/prismbot-web
python3 -m http.server 8080
```

Open: `http://127.0.0.1:8080`

## Deploy options
- Cloudflare Pages
- Netlify
- Vercel (static)
- GitHub Pages
