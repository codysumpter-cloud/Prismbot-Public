# Legacy App Feature Freeze

Effective now, do not add new product features to legacy app folders.
Only allow:
- bug fixes
- security fixes
- migration support changes

All new feature work targets: `apps/prismbot-core`.

Legacy folders under freeze:
- apps/kid-chat-mvp
- apps/mission-control
- apps/prismbot-desktop (backend logic only)
- apps/prismbot-mobile (backend logic only)
- apps/public-chat
- apps/prismbot-site
