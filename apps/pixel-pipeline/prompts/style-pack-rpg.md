# RPG Style Prompt Pack

## Core style constraints
- pixel art, clean clusters, readable silhouette
- limited palette, high contrast edges
- transparent background
- game-ready, top-down or side-view as specified

## Character base prompt template
"{theme} enemy sprite, {size}x{size} pixel art, consistent palette, clean readable silhouette, transparent background, game-ready, {view} view"

## Animation prompts
- idle: subtle breathing/hovering idle loop
- walk: 4-frame walk cycle, readable footwork
- attack: anticipation, strike, recovery
- hit: short recoil
- death: clear collapse/disintegrate
