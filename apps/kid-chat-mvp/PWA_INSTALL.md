# Free Home-Screen App (iPhone)

Use PrismBot as an app on iPhone without App Store/TestFlight.

## Install on iPhone
1. Open the Family app link in Safari.
2. Tap Share (square + up arrow).
3. Tap **Add to Home Screen**.
4. Name it "PrismBot" and tap Add.

## Best experience
- Use HTTPS link (Cloudflare tunnel link is fine).
- Keep one tab/session for account continuity.
- If session expires after backend restart, log in again once.
- Toggle **Eternal local mode** in family chat to run local-primary (server-independent chat/history in browser storage).
- Admin panel includes **Admin Eternal local mode** for server-independent admin chat fallback.

## Current limits vs native app
- No App Store binary install
- Background behavior is browser-limited
- Push notifications require extra setup (not enabled yet)
